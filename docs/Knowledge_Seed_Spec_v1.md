# Humanity Vault AI - Knowledge Seed Specification v1

## Purpose

Knowledge Seeds are portable offline knowledge packages.

They allow users to share knowledge without internet access.

---

# File Extension

.hvs

Humanity Vault Seed

---

# Package Structure

seed.hvs

Contains:

manifest.json

articles/

images/

audio/

videos/

metadata/

---

# Manifest Format

Required:

seed_name
seed_version
author
description
created_date
category
language
article_count
checksum

Example:

seed_name: Survival Seed
seed_version: 1.0
author: Humanity Vault
category: Survival
