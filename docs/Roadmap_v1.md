# Humanity Vault AI Roadmap v1

## Phase 1 - Foundation

Goal:

Build the knowledge platform before AI.

Features:

* Offline Knowledge Library
* Categories
* Search
* Emergency Mode
* Knowledge Seeds

No AI.

## Phase 2 - Enhanced Search

Features:

* Full Text Search
* Semantic Search
* Better Categorization

No AI chat yet.

## Phase 3 - AI Integration

Features:

* Local AI Runtime
* RAG Architecture
* User Selectable AI Models

Supported:

* Qwen
* Gemma
* Llama
* DeepSeek
* Future Models

## Phase 4 - Visual Knowledge Engine

Features:

* Plant Identification Library
* Animal Library
* Tool Library
* Construction Diagrams
* Medical Illustrations

## Phase 5 - Legacy Vault

Features:

* Family Knowledge
* Community Knowledge
* Exportable Knowledge Seeds

## Phase 6 - Civilization Platform

Features:

* Community Contributions
* Multi-Language Support
* Regional Knowledge Packs
* Offline Knowledge Distribution Network
