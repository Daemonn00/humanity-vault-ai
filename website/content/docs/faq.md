# FAQ

**Is Humanity Vault free?**
Yes, free forever &mdash; no ads, no in-app purchases, no subscription.

**Does it require an account?**
No. There is no account system anywhere in the app.

**Does it require an internet connection?**
No, not for the core app. A connection is only needed to browse or
download new Knowledge Packs through Vault Hub, or to visit this
website.

**What platforms are supported?**
Android today. See the [Download](/download/) page for release
status.

**How is a Knowledge Pack verified?**
Each pack declares a verification level per article (`Verified
Source`, `Community Verified`, `Historical Source`, or `Unverified`).
Official packs go through an editorial review process before release.
See [Authoring a Knowledge Pack](/docs/authoring-a-knowledge-pack.html)
for details.

**Where is the source code?**
On [GitHub](https://github.com/Daemonn00/humanity-vault-ai) &mdash;
the app, the Knowledge Packs, the registry tooling, and this website
are all open.

**Can I install a pack that isn't in the official catalog?**
Yes, manually, via **About &rarr; Import Knowledge Pack** in the app.
Only Official Knowledge Packs are listed on this site and in Vault
Hub's in-app catalog.
