# What is Humanity Vault

Humanity Vault AI is a free, offline-first knowledge library for
survival, recovery, and rebuilding. It is built with Flutter and runs
entirely on-device.

## Core principles

- **Knowledge First** &mdash; the app exists to preserve and surface
  practical knowledge, not to sell anything.
- **Offline First** &mdash; the core app has zero network dependency.
  No internet connection is required to install or use it.
- **Free Forever** &mdash; no cost, no ads, no in-app purchases.
- **Privacy First** &mdash; no required account, no tracking.
- **AI Optional** &mdash; any AI-assisted features stay grounded in the
  app's own local articles and Knowledge Packs; they never invent
  survival, medical, security, hunting, or foraging instructions.

## What's included

The app ships with a built-in core library of articles across eleven
categories: Survival, Medicine, Agriculture, Engineering, Electronics,
Construction, History, Philosophy, Islam, Local Wisdom, and
Communication.

Optional **Knowledge Packs** extend that library. Official packs are
listed on the [Knowledge Packs](/packs/) page of this site, sourced
directly from the same published registry the app's own in-app Vault
Hub reads.

## This website

This site is a read-only companion to the app: it introduces the
project, lists official Knowledge Packs, and links to the Android
download once one is published. It has no account system, no backend,
and no way to publish or edit content from a browser &mdash; everything
here is either static or read directly from the published registry.
